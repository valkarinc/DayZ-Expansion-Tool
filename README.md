# 🛠 DayZ Expansion Tool

A powerful, user-friendly utility designed for DayZ server owners to manage Expansion Market configurations and Trader mapping without the headache of manual JSON editing.

---

## 🚀 Download & Installation

To get the latest version of the tool:
1. Go to the [**Releases**](https://github.com/valkarinc/DayZ-Expansion-Tool/releases/) section on the right side of this page.
2. Download the `ExpansionTool_v0.1.zip`.
3. Extract the contents to a folder on your PC.
4. Run `ExpansionTool.exe`.

> [!IMPORTANT]
> This is an unsigned Alpha build. You may need to click **"More Info"** and **"Run Anyway"** on the Windows SmartScreen popup to launch the application.

---

## ✨ Features

### 🛒 Market Editor
* **Types Integration:** Load your `types.xml` to instantly grab classnames.
* **Price Management:** Easily set Min/Max price thresholds.
* **Variants & Attachments:** Nested items (like weapon attachments or skins) can be added with a single click.
* **Customization:** Integrated color picker and icon selection for category headers.

### 📍 Trader Mapper (.map)
* **Automated Formatting:** Generates the exact `Object.Category|Position|Orientation` string required by Expansion.
* **Quick Reference:** Built-in guide for newbies to understand X/Y/Z coordinates and orientation.
* **Error Prevention:** Automatically handles the dot-notation mapping between models and JSON categories.
